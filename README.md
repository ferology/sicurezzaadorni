# Missione Sottosopra · Sicurezza in Internet

Workshop interattivo per bambini di 9 anni sulla sicurezza in internet, ispirato all'estetica di **Stranger Things**.
Realizzato per il **Club dell'Adorni**.

## 🎬 Apri il workshop

👉 **[Avvia la presentazione](https://ferology.github.io/sicurezzaadorni/)**

(Una volta abilitato GitHub Pages — vedi sotto)

## 📋 Contenuto

- **28 slide** in italiano: intro, casi reali, quiz interattivi, slide di riflessione "Diario del Club"
- **Tema Stranger Things**: Hawkins/Sottosopra, Demogorgone, Vecna, Mind Flayer come metafore dei pericoli online
- **Mascotte Dustin** — il nerd-eroe del Club
- **Colonna sonora** Title Sequence (in loop)
- **Intro cinematografica** con zoom su "MISSIONE SOTTOSOPRA" sincronizzato col tema
- Effetti VHS, CRT scanlines, lucine di Natale, vigne del Sottosopra
- 5 quiz interattivi con feedback animato
- 3 casi reali (Sofia, Marco, Giulia)
- 5 slide di riflessione + discussione
- Volantino A4 stampabile con il **Patto del Club** + 5 regole d'oro

## 📁 File

- `index.html` — la presentazione (apri in qualsiasi browser moderno)
- `assets/` — SVG ufficiali Stranger Things + mp3 colonna sonora + varianti colorate
- `Capitan_Pixel_Regole_e_Patto.pdf` — volantino A4 stampabile

## ⌨️ Controlli

| Azione | Tasto |
|---|---|
| Slide successiva | `→` o `Spazio` o click metà destra schermo |
| Slide precedente | `←` o click metà sinistra schermo |
| Musica on/off | Pulsante 🔇 in alto |
| Salta intro | `Escape` o pulsante "SALTA INTRO" |

## 🛠 Stack tecnico

- HTML/CSS/JavaScript vanilla — nessun framework
- Web Audio API per il fallback synth
- Google Fonts: Playfair Display, DM Serif Display, Fredoka, VT323, Caveat
- SVG ufficiali da [svgwaves.io](https://www.svgwaves.io/stranger-things)

## 🌐 Abilitare GitHub Pages

Dopo il primo push:

1. Vai su **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Salva. L'URL pubblico sarà `https://ferology.github.io/sicurezzaadorni/`
